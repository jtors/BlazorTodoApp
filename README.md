## Simple CRUD  App
This is a Simple CRUD app to learn the basics of Blazor

## Tech Stack
- [Blazor](https://dotnet.microsoft.com/en-us/apps/aspnet/web-apps/blazor)

## How to run
```bash
$ npm run install

$ npm run dev

# for another tab or pane
$ dotnet watch
```
Open (localhost)[http://localhost:5255/] in the browser
